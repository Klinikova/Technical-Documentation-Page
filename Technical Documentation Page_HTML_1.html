Who Invented HTML?

  • He came up with the idea of sharing information no matter where a computer was located, through the use of hyperlinks (HTML-coded links that connect one resource to another), HTTP (a communication protocol for web servers and web users) and the URL (a streamlined address system for every web page on the internet).
  • HTML v2.0 was released in November of 1995, after which were seven others to form HTML 5.1 in November of 2016. It's published as a W3C Recommendation.

    • What Does HTML Look Like?

  • HTML tags are written as pairs; there must be a beginning tag and an ending tag in order to make the code display correctly. You can think of it like an opening and closing statement, or like an uppercase letter to start a sentence and a period to end it.
  • The first tag designates how the following text will be grouped or displayed, and the closing tag (signaled with a backslash) designates the end of this group or display.
  • HTML is an application of the (SGML) Standard Generalised Markup Language which is the international standard for Markup. It is a standard format for hypertext documents which includes web pages and alike.
  • Every HTML document adheres to a few rules relating to its structure, these rules dictate the minimum elements required along with their order in the document. Below is the standard element layout:
    • <HTML>
    <HEAD>
    <TITLE> the title of your document <TITLE>
    <META NAME="description" CONTENT="Description of your page goes here">
    <META NAME="keywords" CONTENT="keyphrase 1, keyphrase 2, etc">
    </HEAD>

    <BODY>

    main body of page goes here (content area)

    </BODY>

    </HTML>
  • The <HEAD> element. The HEAD element can contain several different types of data including HTML elements, CSS (Cascading Style Sheet) definitions and also Javascript code.
  • HTML elements which are placed within the HEAD container include the <TITLE> tag and also any of the numerous <META> tags (see below).

    • The <TITLE> element.

    The <META> element.

    <META NAME="description" CONTENT="Description of your page goes here">

    <META NAME="keywords" CONTENT="keyphrase 1, keyphrase 2, etc">

    <META NAME="author" CONTENT="Joe Bloggs">

    <META NAME="robots" CONTENT="noindex">

    <META NAME="language" CONTENT="en">

    The <BODY> element.

    <BODY> content of your page here </BODY>

    How to create a link in HTML

    <A HREF="URL"> link text/image </A>

    <A HREF="#???"> link text/image </A>

    <A HREF="mailto:whoever@example.com"> link text/image </a>

    <A NAME="???"> target text/image </A>
